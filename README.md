🚀 JobList
A Job Management Admin Interface that allows users to manage job listings efficiently. The application consists of a frontend and backend, with APIs deployed on Vercel.

🛠️ Tech Stack
Frontend
React.js
Material UI
Tailwind CSS
Axios (for API requests)
Backend
Node.js
Express.js
MongoDb
CORS enabled
